# SHIELD: A Diverse Clinical Note Dataset for Enterprise-Scale De-identification

> **Dataset coming soon on Stanford Redivis**

## Abstract

Clinical text de-identification is essential for secondary use of electronic health records, yet public benchmarks such as i2b2 2006/2014 are outdated and lack diversity. SHIELD is a diverse dataset of 1,394 clinical notes with 10,505 gold-standard PHI spans across 9 categories, constructed through set-cover diversity sampling from Stanford Medicine with human adjudication. Distributional analysis using Fréchet Text Distance and Jensen-Shannon Divergence confirms that SHIELD occupies a distinct region of biomedical embedding and vocabulary space versus legacy benchmarks. The dataset and distilled DeBERTa v3 model are publicly released to support reproducible research in clinical NLP de-identification.

**Paper:** [arXiv:2605.03301](https://arxiv.org/abs/2605.03301)
